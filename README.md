## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/openai-translator.git
```

Go to the project directory

```bash
  cd ..openai-translator
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
